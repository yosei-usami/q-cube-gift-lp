# Q-CUBE ギフトLP

店頭でQ-CUBEサンプルをスキャンした人に向けた、ギフト訴求のランディングページ。
開店祝い・周年記念などのギフト需要をターゲットに、購入相談を公式LINEへ誘導する。

## 構成
- `Q-CUBE ギフトLP.html` … 本体（11セクション構成）
- `styles.css` … スタイル
- `image-slot.js` / `tweaks-panel.jsx` … 画像枠・微調整UI
- `images/` … 掲載画像
- `preview-mobile.html` … スマホ表示プレビュー

## ローカルプレビュー
```
python -m http.server 8000
```
→ http://localhost:8000/
